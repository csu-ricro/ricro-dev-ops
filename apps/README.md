### Batch Updating Apps
This script updates and rebuilds all apps after an update to the `ricro-app-template`
#### Suggested file structure
```
dev/
  ∟ prod/
  ∟ apps/
    ∟ ricro-apps/
    ∟ ricro-rcr-training/
    ∟ ...
  ∟ dev-ops/
    ∟ ricro-app-template/
      ∟ src/
        ∟ csu-app-template
    ∟ ricro-dev-ops/
      ∟ apps/
        ∟ push-app-template-update.ps1
```
