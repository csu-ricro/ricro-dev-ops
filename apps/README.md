### Batch Updating Apps
This script updates and rebuilds all apps after an update to the `ricro-app-template`
#### Required file structure
```
dev/
  ∟ apps/
    ∟ ricro-apps/
    ∟ ricro-rcr-training/
    ∟ ...
  ∟ dev-ops/
    ∟ ricro-app-template/
    ∟ ricro-dev-ops/
      ∟ apps/
        ∟ push-app-template-update.bat
```
